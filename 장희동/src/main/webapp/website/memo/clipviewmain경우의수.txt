clipviewmain 페이지에서 경우의 수

1. 리플
	1-1 리플 채널 클릭시 해당 채널 페이지로 이동
	1-2 리플 내용 출력
	1-3 리플에 채널명 출력
	1-4 리플 등록(내용만)
2. 영상
	2-1 영상 클릭시 해당 영상이 재생되는 clipviewmain으로 이동
	2-2 영상 제목 출력
	2-3 영상 상세정보(내용)출력
	2-4 영상 썸네일 출력
	2-5 영상 업로드 하는 페이지로 이동
3. 채널
	3-1 채널명 출력
	3-2 팔로우 돼야됨
	3-3 좋아요 돼야됨
4. 댓글
5. 좋아요
